üìã Executive Summary

Product Overview

Quiz Application is a web-based platform with role-based access control featuring JWT authentication. Administrators can create and manage quiz questions, while students take randomly generated quizzes (10 questions per attempt) with instant scoring and feedback.

Key Features

  • ‚úÖ Role-Based Access: Admin manages questions, Student takes quizzes
  • ‚úÖ JWT Authentication: Secure token-based authentication system
  • ‚úÖ Question Management: Admin can add, edit, and delete questions (CRUD)
  • ‚úÖ Random Quiz Generation: Exactly 10 random questions per quiz attempt
  • ‚úÖ Instant Scoring: Automatic grading with immediate feedback
  • ‚úÖ Quiz History: Complete tracking of all quiz attempts and scores

Target Audience

  • Administrators: Teachers/educators creating and managing question banks
  • Students: Learners taking quizzes to test their knowledge
  • Educational Institutions: Schools implementing online assessments

Database Schema

1. Users Table

CREATE TABLE users ( id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(100) NOT NULL, email VARCHAR(100) UNIQUE NOT NULL, password VARCHAR(255) NOT NULL, role ENUM('admin', 'student') NOT NULL DEFAULT 'student', created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP );

2. Questions Table

CREATE TABLE questions ( id INT PRIMARY KEY AUTO_INCREMENT, question_text TEXT NOT NULL, option_a VARCHAR(255) NOT NULL, option_b VARCHAR(255) NOT NULL, option_c VARCHAR(255) NOT NULL, option_d VARCHAR(255) NOT NULL, correct_answer ENUM('A', 'B', 'C', 'D') NOT NULL, category VARCHAR(50) DEFAULT 'General', created_by INT NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, FOREIGN KEY (created_by) REFERENCES users(id) );

3. Quiz Attempts Table

CREATE TABLE quiz_attempts ( id INT PRIMARY KEY AUTO_INCREMENT, user_id INT NOT NULL, score INT NOT NULL, total_questions INT NOT NULL DEFAULT 10, percentage DECIMAL(5, 2) NOT NULL, submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, FOREIGN KEY (user_id) REFERENCES users(id) );

4. Quiz Attempt Answers Table

CREATE TABLE quiz_attempt_answers ( id INT PRIMARY KEY AUTO_INCREMENT, attempt_id INT NOT NULL, question_id INT NOT NULL, user_answer ENUM('A', 'B', 'C', 'D') NOT NULL, is_correct BOOLEAN NOT NULL, FOREIGN KEY (attempt_id) REFERENCES quiz_attempts(id), FOREIGN KEY (question_id) REFERENCES questions(id) );
