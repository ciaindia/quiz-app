üìã Executive Summary

Product Overview

Quiz Application is a web-based platform that enables administrators to create and manage quiz questions while allowing students to take randomly generated quizzes. The system features role-based access control with JWT authentication, ensuring secure and personalized experiences for both administrators and students.

Target Audience

  • Administrators: Teachers, educators, or quiz managers who create and manage question banks
  • Students: Learners who take quizzes to test their knowledge
  • Educational Institutions: Schools and training centers implementing online assessments

Project Goals

  • Enable administrators to efficiently create and manage quiz questions
  • Provide students with randomly generated quizzes (10 questions per attempt)
  • Implement secure authentication using JWT tokens
  • Track quiz attempts and scores for performance monitoring
  • Create an intuitive, user-friendly interface for both roles

Key Features

  • ‚úÖ Role-Based Access: Admin and Student roles with different permissions
  • ‚úÖ JWT Authentication: Secure token-based authentication system
  • ‚úÖ Question Management: Admin can add, edit, and delete questions
  • ‚úÖ Random Quiz Generation: 10 random questions per quiz attempt
  • ‚úÖ Instant Scoring: Automatic grading with immediate feedback
  • ‚úÖ Quiz History: Track all quiz attempts and scores

Role Overview

ADMIN

Primary Function: Question Management

  • Create new quiz questions
  • Edit existing questions
  • Delete questions
  • View all questions in database
  • View question statistics

STUDENT

Primary Function: Quiz Taking

  • Submit quiz answers
  • View immediate results
  • Track quiz history
  • View personal statistics

Admin Role - Detailed Features

Dashboard

  • Total questions count
  • Questions by category/difficulty
  • Recently added questions
  • Quick access to add new question

Question Management

  • Add Question:
    • Question text (required)
    • 4 answer options (required)
    • Correct answer selection (required)
    • Category/Subject (optional)
    • Difficulty level (optional)
  • Edit Question: Modify existing question details
  • Delete Question: Remove question with confirmation
  • View Questions: List all questions with search/filter

Student Role - Detailed Features

    F->>A: POST /api/auth/login
    A->>DB: Query user by email

    A->>A: Compare password (bcrypt)
    A->>A: Generate JWT token
    A-->>F: Return {token, user, role}
    F->>F: Store token in localStorage
    F->>F: Redirect based on role

    DB-->>A: Return data
    A-->>F: JSON response

Quiz Taking Flow

sequenceDiagram
    participant S as Student
    participant F as Frontend
    participant A as API
    participant DB as Database
    
    S->>F: Click "Start Quiz"
    F->>A: GET /api/quiz/random
    A->>DB: SELECT 10 random questions
    DB-->>A: 10 questions
    A->>A: Shuffle questions
    A-->>F: Return questions (without correct answers)
    F->>F: Display first question
    
    loop For each question
        S->>F: Select answer
        F->>F: Store answer locally
        S->>F: Click "Next"
    end
    
    S->>F: Click "Submit Quiz"
    F->>F: Show confirmation
    S->>F: Confirm submission
    F->>A: POST /api/quiz/submit
{questionIds, answers}
    A->>DB: Fetch correct answers
    DB-->>A: Correct answers
    A->>A: Calculate score
    A->>DB: INSERT quiz attempt
    DB-->>A: Success
    A-->>F: Return {score, total, percentage}
    F->>F: Display results
